Projeto de Biblioteca Virtual em Flask

Descrição do Projeto
Este projeto é uma aplicação web de uma biblioteca virtual construída usando o microframework Flask para Python. 
A aplicação permite aos usuários se cadastrarem, fazerem login e visualizarem uma lista de livros disponíveis na biblioteca.

Recursos

Página de Login: Os usuários podem fazer login na aplicação usando suas credenciais.

Página de Registro: Novos usuários podem se registrar na aplicação fornecendo detalhes como nome, e-mail, telefone, cidade e endereço.

Página da Biblioteca: Após o login, os usuários são direcionados para a página da biblioteca, onde podem ver uma lista de livros disponíveis.

Como executar o projeto

Clone o repositório para a sua máquina local.

Navegue até a pasta do projeto via linha de comando.

Instale as dependências necessárias usando o comando pip install -r requirements.txt.

Execute o arquivo app.py usando o comando python app.py.
Abra um navegador web e acesse http://localhost:5000 para visualizar a aplicação.

Dependências
Flask
WTForms (para formulários)

Flask-SQLAlchemy (para banco de dados)
